Les commandes GitHub, basées sur Git, permettent aux développeurs de travailler harmonieusement sur des projets, de suivre les modifications, et de gérer efficacement les branches.

Command Description
git init Initialize a local Git repository
git add . Add all new and changed files to the staging area
git push Push changes to remote repository (remembered branch)
git pull Initialize a local Git repository
